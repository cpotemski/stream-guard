# TabGroup Darstellung

## Entscheidung
- Die verwaltete TabGroup bekommt den kurzen Namen `TW Watch`.
- Wenn von Chrome unterstuetzt, wird die TabGroup-Farbe auf `purple` gesetzt.

## Begruendung
- Der Name bleibt kurz und in der Tab-Leiste gut lesbar.
- `purple` liegt nah an Twitch-Lila und verbessert die visuelle Wiedererkennbarkeit ohne Zusatzkomplexitaet.
