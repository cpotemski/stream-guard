# Entscheidung
Mit Erreichen der v1-Ziele wird die Extension-Version auf `1.0.0` angehoben.

# Begruendung
- Die umgesetzten Funktionen adressieren den v1-Kern (robuste 24/7-Steuerung, geringe Nutzerinteraktion, klare Betriebsuebersicht).
- Der vorherige `0.7.x`-Stand war iterativ; der aktuelle Stand ist als nutzbarer v1-Releasekandidat einzuordnen.

# Scope
- SemVer-Anpassung von `0.7.84` auf `1.0.0`.
- Keine Major-Inkompatibilitaet gegenueber der bisherigen v1-Zielrichtung, aber bewusstes Signal fuer Produktreife v1.
