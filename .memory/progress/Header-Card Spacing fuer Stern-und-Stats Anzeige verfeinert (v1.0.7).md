# Kontext
Nach Integration von Stern+Stats in einer gemeinsamen Card war das visuelle Spacing noch unausgewogen.

# Umsetzung
- Card-Innenabstaende reduziert und ausbalanciert.
- Sternbereich etwas kompakter gemacht.
- Stats-Abstand vereinheitlicht.
- Dezente Trennlinie zwischen Stern und Stats eingefuehrt (nur wenn Stats sichtbar sind).

# Geaenderte Dateien
- `extension/src/styles.css`
- `extension/manifest.json` (Version auf 1.0.7)

# Ergebnis
Die Card wirkt kompakter und gleichmaessiger, ohne Funktionsaenderung.
