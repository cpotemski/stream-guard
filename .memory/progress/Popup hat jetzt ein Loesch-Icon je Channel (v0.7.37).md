# Fortschritt

- Im Popup wurde pro Channel-Zeile hinter den Sortier-Icons (hoch/runter) ein zusaetzliches Loesch-Icon (`🗑`) eingefuegt.
- Klick auf das Loesch-Icon entfernt den jeweiligen Channel direkt aus `importantChannels` und aktualisiert die Ansicht sofort.
- Die vorhandenen Sortier-Buttons und das neue Loesch-Icon nutzen jetzt eine gemeinsame Button-Basis-Klasse (`control-button`) fuer konsistente Darstellung.
- Die Extension-Version wurde gemaess Versionierungsregel auf **0.7.37** erhoeht.

# MVP-Relevanz

- Ermoeglicht schnelleres Aufraeumen der Streamliste direkt im Popup ohne Zusatzdialoge.
- Bleibt innerhalb des bestehenden MVP-Scopes (Priorisierung/Verwaltung wichtiger Channels).
